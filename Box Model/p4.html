CSS BOX MODEL..!

every element that can be displayed on a web page is comprised of one or more rectangular boxes.css box model typically describes how these rectangular boxes are laid out on a web page. these boxes can have different PROPERTIES AND CAN INTERACT WITH Each other in different ways, but every box has a content area and optional suurounding padding, border , and margin areas.
